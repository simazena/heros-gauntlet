# Hero's Gauntlet

A third-person wave-based combat arena built in Unity 6. You play an enhanced hero placed inside the Gauntlet, an arena designed to test superhuman fighters. Survive escalating waves of fast brawlers and heavy brutes to complete the trial.

## Features
- Punch (3-hit) and kick (2-hit) combos
- Roll and jump-over evades with invulnerability frames
- Three difficulty levels (Easy, Normal, Hard) affecting enemy health and damage
- Wave spawner with banner announcements and health pickups between waves
- IMGUI HUD: player and enemy healthbars, low-health vignette, story intro
- Sound effects for hits, misses, pickups, deaths, and victory

## Controls
| Input | Action |
|-------|--------|
| W A S D | Move |
| Mouse | Look |
| Shift (hold) | Sprint |
| Space (tap) | Jump |
| Space (hold) | Jump Over (invulnerable) |
| Left Ctrl | Roll (invulnerable) |
| Left Click | Punch combo |
| Right Click | Kick combo |

## Running the Project
1. Open the project folder in Unity 6 (URP).
2. Open `Assets/Scenes/SampleScene.unity`.
3. Press Play.

## Build
File -> Build Settings -> select target platform -> Build.
